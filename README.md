# Database Writer Common [DBWC]

Common classes for creating vendor specific database writers.

## Installation
Install via composer:

```bash
composer require db-writer-common
```

composer.json
```json
{
    "require": {
      "db-writer-common": "^6.0"
    }
}
```

## Development

Run tests:
```bash
docker-compose run --rm tests
```
